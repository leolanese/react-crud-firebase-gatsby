### React CRUD

Implemented CRUD App using: React (Hooks), Gatsby (SSR), Firebase (dB), JS, CSS and HTML.

### Deployment Process:
```javascript
git clone https://github.com/leolanese/react-crud-firebase-gatsby.git
cd react-crud-firebase-gatsby
npm install
gatsby develop

open http://localhost:8000
```
